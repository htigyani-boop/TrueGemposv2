Termux Server Setup Guide
On the Android phone connected to printer:

1. Install Termux from F-Droid (free)
2. Open Termux and run:
pkg update -y && pkg install nodejs -y
mkdir printserver && cd printserver
npm init -y
npm install express cors escpos escpos-usb
node server.js
3. Upload server.js (download below) to that phone
4. Note the IP shown in Termux — enter it above
